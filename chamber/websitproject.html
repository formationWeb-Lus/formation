Website Plan - Web Development Courses Platform

Site Name

Site Name: creawebcourses.com

Reason for the Name: "reawebcourses.com" is simple and clear, reflecting the purpose of the site: to help users learn web creation, development and programming. "Courses" indicates that this is a platform dedicated to training.

Suggested Domain: creawebcourses.com (available, as an example).

Site Purpose

The purpose of this site is to provide a comprehensive platform where users can discover, promote, and enroll in web development and programming courses. Users will be able to explore training programs based on their skill level, technology of interest (such as HTML, CSS, JavaScript, Python, etc.), and preferred course format (online courses, workshops, bootcamps, etc.).

Scenarios

  • How can I compare multiple web development courses to choose the best one for me?
  • What are the best courses for learning Python at a beginner level?
  • Where can I find in-person workshops for learning JavaScript?
  • How do I leave a review for a course I've taken?

Color Scheme

Color 1: Green (#4CAF50) - used for headings and action buttons such as "Sign Up" or "Compare".

Color 2: Orange (#FF5722) - used for subtitles, accents, and important notifications.

These colors were chosen to create a strong contrast while maintaining a welcoming and professional appearance.

Typography

Font 1: Arial (used for body text, descriptions, and menus).

Font 2: "Roboto" (used for headings and subheadings to provide a modern and readable look).

Wireframe

Here is a sketch of the homepage layout, including both desktop and mobile versions.

Wireframe - Desktop Version

  • Header: Logo, navigation menu (Home, Courses, Compare, Blog, Forum, About)
  • Main section: Banner with introductory text and call-to-action to explore courses.
  • Secondary section: List of popular courses with filters to search by level, technology, and course type.
  • Review section: Display of the latest reviews left by users on courses.
  • Footer: Legal information, social media links, contact page link.

Wireframe - Mobile Version

  • Hamburger menu with access to main sections.
  • Image and introductory text in a card format.
  • List of courses with swipeable filtering options.
  • Review section with horizontal scroll to view recent comments.
  • Compact footer with social icons and essential links.

CSS

The CSS used in this plan applies the colors and typography outlined above. Elements such as buttons, menus, and links will be styled to be visible and accessible across all platforms.
